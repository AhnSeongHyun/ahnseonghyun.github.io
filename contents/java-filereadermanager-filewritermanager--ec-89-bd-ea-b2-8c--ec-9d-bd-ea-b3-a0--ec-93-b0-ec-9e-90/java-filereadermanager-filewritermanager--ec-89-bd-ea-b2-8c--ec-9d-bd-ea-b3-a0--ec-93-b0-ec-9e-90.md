<span style="font-size: 11pt; ">자바에서 파일 읽고 쓰는 부분에 대한 정리차원에서 쓴다. 개인적으로 자주 쓰는 FileReaderManager, FileWriterManager 클래스를 올린다. 유틸리티 클래스(static  함수)로 중요한 부분은 현재 JVM 인코딩을 가져와서 읽고 쓰고 하는 부분이다.
