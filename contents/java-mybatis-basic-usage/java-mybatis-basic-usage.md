– mybatis-3.1.1.jar

– mysql-connector-jara-5.1.22-bin.jar

**db.propertise 작성**

– 데이터베이스 연결 정보가 담긴 파일 생성

– 아래의 내용은 mysql을 대상으로 작성된것임

