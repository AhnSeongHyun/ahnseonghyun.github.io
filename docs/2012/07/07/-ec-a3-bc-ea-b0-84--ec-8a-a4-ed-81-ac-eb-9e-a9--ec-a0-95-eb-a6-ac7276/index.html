time 실행명령어 

이렇게 입력하면 실제 해당 프로그램에 대한 실행 시간을 커널 레벨과 유저 레벨로 확인 할 수 있습니다. 장점이라면 단연 코드안에 time_t 와 같은 시간 측정을 위한 코드를 쓸 필요가 없다는 것이겠죠?
