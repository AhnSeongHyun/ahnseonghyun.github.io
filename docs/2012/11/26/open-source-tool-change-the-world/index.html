오픈소스 도구가 어떻게 민주주의를 발전시킬 수 있을까?

Clay Shirky GitHub open-source ted 어떻게 인터넷이 정부를 완전히 바꾸어 놓을것인가?

윤석찬님의 트윗을 통해서 알게된 TED 동영상. GitHub에 대한 내용이 나온다. 기본적으로 문서에 대한 버전관리를 소스버전관리를 하는 github를 이용하는 모습들이 그려지고 있다. 이런게 진짜 사회적 기여가 아닌가 싶다. 의도했던 안했건 간에 github 는 이미 사회적기여를 하고 있는 것이다.
