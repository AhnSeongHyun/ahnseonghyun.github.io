네트워크 기반 지능형 로봇 시스템 설계 및 구현

Project Organization : Konkuk Univ. CSIR
Duration : 2005/11 ~ 2006/10
Work : Development
Language : Visual C++, MFC
DBMS : SQL2000
Using S/W : Visual C++ 6.0
Using H/W : Hanwool Robotics, HP Ipaq hx7400
Task
로봇의 위치 확인을 위한 PDA 프로그램
 – 로봇의 초음파 센서로 부터 들어오 데이터가 로봇 자체의 무선랜을 통해서 서버의 데이터베이스로 전송 
 – 데이터베이스 내의 로봇의 좌표데이터를 이용해서 로봇의 이동에 따른 로봇 주위의 물체의 위치를 PDA
   에 지도의 형태로 나타냄으로써, 사용자가 언제든지 로봇의 움직임을 파악할 수 있음. 
 – 간단한 조작을 통해서, 로봇이 교착상태에 빠졌을때 사용자가 로봇을 제어 할수 있음.
