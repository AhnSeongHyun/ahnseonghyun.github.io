2010-07-21

Project Organization : Yonsei.Univ LOMIIO

Work : Development

Language : .NET 2.0 with C#

DBMS : SQL2005, Windows Server 2003

Using S/W : Visual Studio 2005
Task

1. Body Signal Display and Transmission Program
    Duration : 2008/01 ~ 2008/03
    Using H/W : KTMED Oxyplus/KTPM, 세인전자 혈압계


– Data Acquisition & Display from Medical Device(ECF, SpO2, BP)
– Data 전송을 위한  HL7(Health Level 7) 메시지 변환과 전송


– 각종 의료기기(혈당계, 혈압계, ECG, SpO2)로 부터 생체 신호 데이터를 시리얼 통신을 통해서
  PC로 전송 받고, 전송 받은 신호를 HL7 V3 표준에 맞게 XML 문서 형식으로 변환해서 서버에 있는
  데이터베이스에 저장하는 기능을 하는 시스템 개발

– XML 문서 전송과 데이터베이스 저장을 위해서 웹 서비스(Web Service) 기술 사용. 

2. UBDC Tcp/IP Server 
    Duration : 2008/07
    Using H/W : Belt형 ECG 기기 
– 벨트형 ECG 기기에서 이동중에 측정한 ECG 데이터를 전송받기 위한 서버
– ECG 데이터가 CDMA 통신을 통해서 서버로 전송됨. 
– 다수의 벨트형 클라이언트를 지원할 수 있도록 설계 : 다중 접속 서버 
– 전송받은 ECG 데이터를 HL7 aECG(Annotated ECG) 메시지 형태로 변환
– 웹 서비스를 이요해서 해당 메시지를 파싱(Parsing) 후, 데이터베이스에 ECG 데이터 저장
3. Alarm System for Abnormal Bio-signal(이상 샌체신호 데이터에 대한 알람 시스템 개발)
    Duration : 2008/08
    Using H/W : xonda .NET SMS Dll
– 측정 생체신호 데이터의 이상 징후 발견시, SMS 서비스
– 사용자가 측정한 생체 신호를 입력시, 데이터 필터링을 통해서 이상 데이터 산출
– 이상 데이터 발생시 확인 요청 메시지를 주치의 에게 SMS로 통보 
– SMS 전송에 대한 기록은 History DB에 저장
