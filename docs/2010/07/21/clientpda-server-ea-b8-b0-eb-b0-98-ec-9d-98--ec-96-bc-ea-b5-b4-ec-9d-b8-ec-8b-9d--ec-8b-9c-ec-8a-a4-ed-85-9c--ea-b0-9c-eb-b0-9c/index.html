Project Organization : Konkuk Univ. Personal
Duration : 2006/03 ~ 2006/11
Work : Development
Language : Visual C++, MFC
DBMS : MSAccess
Using S/W : Visual C++ 6.0
Using H/W : Samsung Anycall MITS 4300
Task
Client(PDA) – Server 기반의 얼굴인식 시스템 
 – PDA 폰에 장착된 카메라를 이용해서, 상대방의 얼굴을 찍은후 제작된 프로그램을 이용해서 
   원격지의 서버에 얼굴 사진을 전송. 
 – 사진을 받은 서버에서 이미지 프로세싱을 통해서 각 얼굴의 특징을 밝기의 평균값을 추출해내고,
    데이터베이스 내에서 밝기의 평균값에 가장 유사한 사람을 찾아낸다. 
– 찾은 사람에 대한 환자 개인 정보와 질병관련 정보를 다시 PDA 폰으로 전송.
– Problem : 조명에 따른 밝기의 민감도 문제있음.
