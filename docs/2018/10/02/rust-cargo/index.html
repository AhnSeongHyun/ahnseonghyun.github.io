빌드 및 실행

  • cargo build: 컴파일
  • cargo run : 바로 실행
  • cargo check : 문법체크
  • cargo clean : 컴파일 정리
  • cargo new : 프로젝트 생성
  • cargo init : 프로젝트 생성(이미 존재하는 디렉토리에 생성 가능)
  • cargo test : tests 실행
  • cargo bench : 벤치마크 실행
  • cargo update : 의존성 업데이트
  • cargo search : crates 검색 

$ cargo search web

  • cargo publish : 패키징 및 해당 프로젝트 등록
  • cargo install : Rust binary 설치
  • cargo uninstall: Rust binary 제거
