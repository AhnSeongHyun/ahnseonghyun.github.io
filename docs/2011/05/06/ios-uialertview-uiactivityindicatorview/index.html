[iOS] UIAlertView + UIActivityIndicatorView

dev iOS UIActivityIndicatorView UIAlertView 로딩뷰




한우찾기 앱을 만들면서 사실 자바스크립트를 통해서 데이터를 긁어오는 과정을 시간을 측정해 보니, Wi-Fi 사용시에는 사용자가 느리다고 느끼지는 않지만, 3G를 사용할때는 그렇지가 않더라구요. 그래서 로딩뷰를 하나 구성하게 되었습니다. 이미 앞선 <로딩뷰 관련 포스팅> 에서 말했던 것 처럼, 원래는 echofon 형태의 로딩 스타일을 쓰다가 별루 이쁘지 않아서 바꾸게 되었는데요. 그게 바로 UIAlertview 에 UIActivityIndicatorView 붙여서 쓰는 것인데요. 아래의 사진처럼 나오게 됩니다.
