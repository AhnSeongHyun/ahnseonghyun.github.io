카테고리(Category)

– 어떤 클래스에 기능을 추가하는 방식 

– 상속(inheritance)를 사용하지 않고, 어떤 클래스에 메소드만을 추가하는 방식

– 상속과 다르게 변수는 추가할 수 없고, 메소드만 추가 가능함. 

사용법

파일명

– 기존클래스+확장이름

– ex) NSString+Dollar, NSString+LongText

코드

호출

– 호출시에는 확장 클래스를 import 한 상태에서 기존 클래스로 객체를 만들고 추가한 함수를 만들면 된다.
