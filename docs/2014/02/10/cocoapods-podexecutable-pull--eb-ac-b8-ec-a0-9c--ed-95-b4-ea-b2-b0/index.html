(cocoapods) [!] Pod::Executable pull 문제 해결

CocoaPods Pod::Executable pull cocoapods error dev ios-development

어느순간 cocoapods를 사용하다 보면, pod install 이라고 쳤는데 console 창에 빨간 글씨의 화면이 아래처럼 나올때가 있다. 이러면 pod install 로 인해서 workspace가 생기지 않게 된다. 대처법은 간단한데 cocoapods의 master repo 를 지우면 된다.  

$sudo rm -rf ~/.cocoapods
