Internet Explorer 켜자마자, 작동이 중지되었습니다. 오류 해결 방법

2013 연말정산 dev Internet Explorer 8 Internet explorer 에러 Internet explorer 작동이 중지되었습니다.

한동안 인터넷 익스플로러가 회사 컴퓨터에서 키자마자 작동이 중지 되었습니다. 라는 팝업과 함께 꺼지는 경우가 많았는데 이번 연말정산을 하기 위해서 마음 먹고 지울 각오하고, 컴퓨터 밀 각오하고 고쳐보기로 했다. 

여러가지 검색 결과가 있었지만, 필자가 확인해서 테스트 한 결과중 좋은것은 네이버 지식인에 있는글이었다. 요지는 무엇이냐 하면, 인터넷 익스플로러를 지우거나 재 설치 하지 않고 고칠수 있는 방법인데, Microsoft Fixit 을 사용하는 방법이었다. 

[[지식인 링크](http://kin.naver.com/qna/detail.nhn?d1id=1&dirId=106&docId=156365173&qb=aW50ZXJuZXQgZXhwbG9yZXIg7J6R64+Z7J20IOykkeyngOuQmOyXiOyKteuLiOuLpA==&enc=utf8§ion=kin&rank=2&search_sort=0&spq=0&pid=RnOaBc5Y7t0ssbKNbqKsssssss4-415489&sid=UPdFi3JvLDsAAAHYC1Y)] [[파일 다운로드 링크](http://istqb.tistory.com/attachment/cfile22.uf.173BB3164B90B1B551860C.msi)] 

해당 파일을 실행하면 인터넷 익스플로러 내에 개인적인 데이터, 쿠키등을 지울것이냐고 물어 보는데 지운다고 동의하면 자기가 알아서 지워준다. 그리고 나서 다시 켜 보면 인터넷 익스플로러가 제대로 작동되는것을 확인 할 수 있었다. 

연말 정산의 계절이다. 인터넷 익스플러로 안된다고 옆 동료 컴퓨터에서 하거나 윈도우 서버에 원격접속 해서 하는 일은 없도록 잘 대비하자.^^
